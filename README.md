## 📌 Scablible Backend Habit Tracker

## 🚀 Features

 Add, edit, and delete habits and store curent habits into a data table.

## 🛠️ Tech Stack
Language: Java 24

Frameworks: Spring Boot

Database: PostgreSQL

Build Tool: Maven 

IDE: IntelliJ IDEA

Containerization: Docker Compose

Endpoint Testing: Postman
